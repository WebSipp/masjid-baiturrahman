Petunjuk Penggunaan:

  • Display Digital: Buka di browser yang akan ditampilkan di TV/layar masjid. Gunakan mode fullscreen.
  • Admin Panel: Buka di komputer/HP untuk mengelola konten.
  • Pastikan koneksi internet tersedia untuk sinkronisasi data.
  • Untuk tampilan optimal, gunakan browser Chrome atau Edge versi terbaru.
